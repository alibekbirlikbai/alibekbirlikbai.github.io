Pastebin
API сервис

Описание: Pastebin – это приложение в котором пользователи могут создавать блоки текста, сохранять их в системе, и делиться ссылками на этот текст с другими пользователями по Сети.

Документация: https://github.com/alibekbirlikbai/Pastebin-backend

Stack:

Java (8+), Spring Boot, Spring MVC, PostgreSQL;
